L'idée de ce générateur est de permettre la génération du pdf à partir d'un examen défini avec asciidoc
Vous écrivez votre examen à l'aide d'un éditeur asciidoc comme asciidoclive. Vous téléchargez le résultat en html. Vous chargez ce fichier html sur cette application. Vous imprimer en pdf depuis votre navigateur (Ctrl +P)


Powered by paged.js.

Comment fabriquer son examen

Utilisez asciidoclive pour créer votre énoncé d'examen.
Télécharger votre énoncé en html.
Charger le fichier html sur ce site.
Imprimer votre énoncé (Ctrl + P ou Cmd + P).

Asciidoc est fun Fabriquer votre énoncé devient ludique

In your asciidoc, you can directly include svg like the following to create your box for the answer. The width of a page is 600px. The height is max at 800px. 

pass:[<div align="center"><svg height="200" width="600">
<rect style="fill:rgb(255,255,255);stroke-width:1;
 stroke:rgb(0,0,0)" height="200" width="600"></rect>
 </svg></div>]

Fabriquer des QCMs devient simpleUtilsation de la notion de tâche

					* [ ] good
					* [ ] bad
					* [ ] ...
